## 简介 / About

我是 **许成葳 (Chengwei Xu)**，工业设计本科出身，现在在德企担任技术工程师。

主要负责全球品牌零售展示项目，包括 **雀巢 (Nespresso)**、**IWC**、**HUGO BOSS** 等品牌的工程深化、生产协调与出口交付。项目遍及 20+ 国家，从澳洲到北美、从日本到中东。

## 设计特点 / Design

- **暗色赛博朋克风** — 深紫色主色调，氛围感渐变背景
- **Canvas 环状光环** — Hero 区域右侧的动态波动光环，脉冲呼吸效果
- **毛玻璃弹窗** — Projects 详情以玻璃质感卡片展示，半透明模糊背景
- **响应式布局** — 桌面/平板/手机全适配
- **双语内容** — 中英文混排，切换自然

## 技术栈 / Tech Stack

- Vanilla HTML / CSS / JS（无框架）
- Vercel（托管 & 部署）
- GitHub Pages（备份源）
- Canvas API（环状光环动效）
